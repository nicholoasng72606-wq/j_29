# JUPAS Calculator

This is a website which is designed for students who are studying Physics, Chemistry, Biology, or M2.
The website will help you to calculate your scores to assist you to make decisions in choosing JUPAS programmes.

<a href="https://nicholoasng72606-wq.github.io/j_29/" target="_blank">Click me to open the calculator 🔗</a>
---

## Features
- 🎓Support specific programmes offered by HKU, CUHK, HKUST, PolyU
- 📱Calculate the score automatically by using different formulas for each programme
- 🖊️Able to input your own scores 
- 🏷️Show different colour-coded admission chances
- 📋Filter different programmes by category

## How to use
1. Open the website
2. Input your scores of each subject using the dropdown menus
3. Your calculated scores and admission chances will appear in the table automatically
4. Use the **Category** dropdown to filter programmes by discipline
